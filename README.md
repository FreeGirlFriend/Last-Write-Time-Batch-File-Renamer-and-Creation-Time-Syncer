# 文件批量命名器

基于 EXIF 信息或文件修改时间，批量将文件重命名为 `yyyy_mm_dd_hh_mm_ss.xxx` 格式。

- 图片文件优先读取 EXIF `DateTimeOriginal`
- 其他文件使用修改时间
- 重命名后自动同步创建时间 = 修改时间
- ttkbootstrap 现代 UI 界面

## 环境要求

- Python 3.8+
- Windows / macOS / Linux

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/文件批量命名器.git
cd 文件批量命名器

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行
python 图片命名器.py
```

Windows 用户也可以直接双击 `启动.bat`。

## 使用说明

1. 点击 **浏览...** 选择文件夹
2. 「扩展名」默认 `*` 会扫描所有文件；也可以填写 `.jpg .png .mp4` 只扫描特定类型
3. 点击 **扫描文件**，预览重命名结果
4. 在表格中检查，右键可移除不想处理的文件
5. 点击 **执行重命名**，确认后一键完成

## 命名规则

| 场景 | 时间来源 | 示例 |
|------|---------|------|
| 有 EXIF 的图片 | DateTimeOriginal | `2024_06_15_14_30_22.jpg` |
| 无 EXIF 的文件 | 文件修改时间 | `2025_03_10_18_45_00.pdf` |
| 同秒冲突 | 自动追加 `_1`, `_2` | `2024_06_15_14_30_22_1.jpg` |

## 技术栈

- Python 3
- ttkbootstrap (现代 UI 主题)
- Pillow (EXIF 读取)
- tkinter (内置 GUI)
- Win32 API `SetFileTime` (Windows 创建时间同步)
